LinkedList:
La sintaxis en el main es 
LinkedList<"objeto"> A = new LinkedList<"objeto">
los metodos de A para agregar "objeto"s son:
A.add("objeto")
A.addFirst("objeto")
A.addLast("objeto")

para manejar la LinkedList:

public static void mostrar(LinkedList<Tema> A)
    {
        ListIterator<Tema> r=A.listIterator(); // para lista de objetos

        while(r.hasNext())
        {
                Tema tx = r.next();
                tx.mostrar();//proceso
        }
    }
